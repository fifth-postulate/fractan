## Infinitude of the primes

> There are an infinite number of primes

--

\\(p_1, p_2, \ldots, p_k\\) e.g. \\(2, 3, 5\\)

--

\\(p_1 \cdot p_2 \cdot \ldots \cdot p_k + 1\\) e.g. \\(2 \cdot 3 \cdot 5 + 1 = 31\\)

--

This number has a prime divisor \\(q\\) that differs from the known primes. E.g. \\(31\\)

## First Insight

> The state of a Fractan program is a bank of registers
>
> Each register holds a non-negative number
>
> Each register is indexed by a prime number
>
> There are only finite number of non-zero registers at each moment

## Second Insight

> Each instruction of a Fractan program operates on the bank of registers
>
> an instruction 
>
> * **subtracts** the prime exponents in the _denominator_ from the corresponding prime exponents of the state
> * **adds** the prime exponents in the _numerator_ to the corresponding prime exponents of the state
